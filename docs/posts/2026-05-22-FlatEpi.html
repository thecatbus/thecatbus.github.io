Affine subsets
Posted on May 22, 2026 by Parth
(geometry, algebra)

The structure sheaf \(\mathscr{O}_X\) of a scheme \(X\), upon taking inverse image, enhances any collection of points \(U\subset X\) with the structure of a locally ringed space. The most tame and familiar instance of this is when \(U\) is the space underlying an open subset of \(X\), where the corresponding locally ringed space is simply an open subscheme. What subsets of an affine scheme are affine? Based on [[Jan Šťovíček]]’s article.

There are also affine open subsets not of this form.

Example.1 Take an elliptic curve \(E\subset \mathbb{P}^2\) intersecting \(\{z=0\}\) in the divisor \(3p\), so that \(X=E\smallsetminus \{p\}\) is affine. For any \(q\in X\), the open subvariety \(X\smallsetminus \{q\}\) is affine (since \(np+nq\) is very ample on \(E\) for \(n\gg 0\)). But it is not always a distinguished open subset cut by a non-zero divisor in \(\mathbb{C}[X]\).

Indeed if \(q=\mathbb{V}(f)\) for some \(f\in \mathbb{C}[X]\), then some multiple of \(q-p\) is the principle divisor \((f) \in \text{Div}(E)\), so that \(q-p\) is a torsion element in \(\text{Pic}^0(E)\). But there are many choices of \(q\) for which \(q-p\in \mathop{\mathrm{\text{Pic}}}^0(E)\) is torsion-free!

The correspondence also accounts for subsets that aren’t open at all – if \(\mathfrak{p}\subset A\) is a prime ideal, then we have the flat epimorphism \(A\to A_\mathfrak{p}\) which defines the subset \(\{\mathfrak{q}\in \mathop{\mathrm{\text{Spec}}}A\;|\; \mathfrak{q}\subset \mathfrak{p}\}\). This contains the generic points of all irreducible closed subschemes that generically pass through \(\mathbb{V}(\mathfrak{p})\).

Precisely characterising the collection of subsets \(\mathcal{U}\subset 2^{\mathop{\mathrm{\text{Spec}}}A}\) defined by flat epimorphisms is thus a subtle problem.

The set \(\mathcal{U}\) is known to be closed under arbitrary intersections. We also have some necessary conditions it must satisfy:

  1. Any set \(U\in\mathcal{U}\) must be the intersection of quasi-compact Zariski opens, i.e. its complement must be a Thomason set. If \(A\) is Noetherian, this is equivalent to being closed under generalisation.

  2. If \(A\) is Noetherian, we have a version of Serre’s criterion for affine-ness: \(U\in \mathcal{U}\) if and only if it is generalisation-closed and the local cohomology groups \(\text{H}^{>1}_{V}(X,\mathscr{O}_X)\) at \(V=X\smallsetminus U\) vanish2. It follows that each minimal prime in \(X\setminus U\) must have height \(\leq 1\), i.e. \(X\smallsetminus U\) (and hence \(U\)) is determined the collection of irreducible codimension \(\leq 1\) subvarieties (i.e. connected components and prime divisors) generically inside it.

If \(X=\mathop{\mathrm{\text{Spec}}}A\) is connected and Noetherian, non-trivial flat epimorphisms out of \(A\) thus furnish bipartitions of the set \(\text{Prime}(X)\) of the prime divisors in \(X\). For example any multiplicatively closed subset \(S\subset A\) determines a partition rule “\(Y\notin U\) if and only if some \(f\in S\) generically vanishes on \(Y\)”, and the corresponding flat epimorphism is precisely the localisation \(A\to S^{-1}A\).

If \(X\) above is locally factorial, this injection \(\mathcal{U}\setminus \{X\}\hookrightarrow 2^{\text{Prime}(X)}\) is in fact a bijection, i.e. each subset of \(\text{Prime}(X)\) determines a flat epimorphism. The same holds when \(X\) is a connected curve, the set of non-trivial flat epimorphisms is in bijection with the powerset on closed points of \(X\).

In general the condition is not sufficient.

[!example] Example3 Writing \(Z=\mathop{\mathrm{\text{Spec}}}\mathbb{C}[[x,y]]\) for the formal neighbourhood of the origin in \(\mathbb{A}^2\), note that \(X\setminus \{0\}\) cannot support a flat epimorphism by the height criterion.

We bootstrap this by stretching \(\{0\}\) into a prime divisor – in the formal neighbourhood \(\mathop{\mathrm{\text{Spec}}}\mathbb{C}[[x,y,z]]\) of the origin in \(\mathbb{A}^3\), consider the surface \(\mathbb{V}(xz)\), i.e. the union \(X\cup Z\) of the \(y\)-\(z\) and \(x\)-\(y\) planes. The \(z\)-axis \(\mathbb{V}(x,y)\) is a prime divisor in \(\mathbb{V}(xz)\), but we show that \(U=\mathbb{V}(xz)\smallsetminus \mathbb{V}(x,y)\) cannot arise from a flat epimorphism – indeed if it did, then the base change to (i.e. intersection with) \(Z\) would arise from a flat epimorphism, but this doesn’t happen.

Stretching up a dimension again, consider the formal \(cA_1\) singularity \(\mathbb{V}(xy-zw)\subset \mathop{\mathrm{\text{Spec}}}\mathbb{C}[[x,y,z,w]]\). The divisor \(\mathbb{V}(x,y)\) does not have affine complement – the proof is analogous, since intersecting with \(\{w=0\}\) leaves us with the above case of a line in two intersecting planes.
